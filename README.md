# Hyundai Mobis (hyundai-mobis)

Hyundai Mobis is a South Korean global Tier 1 automotive supplier headquartered in Seoul. Part of the Hyundai Motor Group, Mobis designs and manufactures chassis, cockpit, and front-end modules, plus electrification, ADAS, and infotainment systems for global automakers.
